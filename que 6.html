Programming Languages

Python
A high-level, interpreted programming language known for its readability and versatility.
JavaScript
A versatile scripting language primarily used for web development, both on the client and server side.
Java
An object-oriented, class-based, and concurrent programming language widely used for building enterprise-scale applications.

Web Technologies

HTML
The standard markup language for creating web pages and web applications.
CSS
A style sheet language used for describing the look and formatting of a document written in HTML.
React
A JavaScript library for building user interfaces, particularly for single-page applications.
